* Org Notes

  - Shift-TAB   open/close all headings
  - bullets:  see init.el, org-bullets
  - C-c-[  Add to Agenda List
  - Today's date:    C-c .       (Ex: <2026-03-24 Tue>)
  - Schedule:   C-c C-s
  - Deadline:   C-c C-d
  - Better outline:  Org Indent mode (M-x org-indent-mode RET),
  - C-c a t Display all TODOs

*** To link:
1. This will be link:  [the link][description]
2. Next, surround it by another pair of brackets [
3. Ex:   [https://nytimes.com][nytimes]   (Remember to enclose in [)
4. To Edit existing link:  C-c C-l
5. To create a link to HEADING called HEAD
6. type : [*HEAD]  and surround with another set of brackets [
